## 📌 Tentang Project

Project ini merupakan aplikasi **Analisis Jejaring Sosial (Social Network Analysis / SNA)** berbasis Python yang digunakan untuk menganalisis hubungan antar node dalam sebuah jaringan.

Aplikasi ini dapat membaca data jaringan, membangun graf, menghitung nilai centrality, serta menampilkan hasil analisis dalam bentuk visualisasi.

---

## 🎯 Tujuan

- Menganalisis pola hubungan dalam sebuah jaringan.
- Mengetahui node yang memiliki pengaruh terbesar.
- Mengidentifikasi hubungan antar aktor dalam jaringan.
- Menampilkan visualisasi graf agar mudah dipahami.

---

## 🚀 Fitur

✅ Import data jaringan dari CSV  
✅ Membuat Graph menggunakan NetworkX  
✅ Perhitungan:

- Degree Centrality
- Betweenness Centrality
- Closeness Centrality

✅ Visualisasi jaringan  
✅ Dashboard berbasis web  
✅ Menampilkan hasil analisis secara interaktif
